# traditional_dex_volumes

## Files and Folders Overview

### dex_names_volume_change.py

- Returns name of DEXs, their display names, and change in volume over 1-day, 7-day, and 30-day periods.
- Change cells represent % change based on time period. For example -32.15 is the equivalent of -32.15% change over the specified period.
- This file should be regenerated on a semi-regular basis to capture all new DEXs added to the API.
- See [here](https://github.com/check-sked/crypto_data_resources/blob/main/csv_examples/dexs/dex_names_and_volumes.csv) for example of file returned.
